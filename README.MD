## 使用
头部菜单分为头部左菜单和头部右菜单。需要分别在 外观>菜单 里新建两个菜单，分别添加菜单结构后选择显示位置为头部左菜单和头部右菜单。

主题提供了四个放小工具的侧边栏，为了好看，我建议每个侧边栏放一个小工具。

ICP备案号请在 Settings>常规 里设置ICP备案号。
